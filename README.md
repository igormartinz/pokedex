# Projeto Pokédex

Uma versão alternativa da Pokédex desenvolvida consumindo a PokeAPI, que permite pesquisar, visualizar estatísticas, tipos, evoluções e outras informações do pokémon.

## Tecnologias Utilizadas

- HTML5 - estrutura semântica
- CSS3 - estilização
- Javascript - lógica e consumo de API
- Git e GitHub - versionamento
- Vercel - deploy da aplicação

## Funcionalidades

- Busca Pokémon por nome ou ID
- Navegação entre Pokémons
- Exibição de sprite animada
- Exibição de informações sobre o Pokémon

## API Utilizada

O projeto utiliza a PokeAPI para obter as seguites informações dos Pokémons:

- ID
- Nome
- Imagem
- Altura
- Peso
- Tipo
- Evolução
- Estatísticas

API: [PokéAPI](https://pokeapi.co/)  
Endpoint base: `https://pokeapi.co/api/v2/`
